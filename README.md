# 属性とその評価に着目した映画関連度の表現手法（卒業研究）

## モジュール一覧
- movie.py : 映画情報のスクレイピング
- review.py : レビューのスクレイピング
- sent_dic.py: 極性辞書の整形
- word_cluster.py : 単語ベクトルをクラスタリング
- analyze.py : レビューの解析
- calc.py : 関連度の計算
